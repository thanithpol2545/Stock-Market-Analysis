# Glossary — Key Terms A-Z

## A

**Altman Z-Score** — Multivariate bankruptcy prediction model developed by Edward Altman (1968). Z < 1.81 = distress. See [[Financial-Health/altman-z-score]].

**Asset-Based Valuation** — Valuation method that sums the fair value of all assets minus liabilities. Used for distressed/Liquidation. See [[Research-Methods/lbo-and-sotp]].

## B

**Beta (β)** — Measure of systematic risk. β = 1.0 means the stock moves with the market. β > 1.0 = more volatile. See [[Valuation-Core/capm-model]].

## C

**CAPM** — Capital Asset Pricing Model: Kₑ = Rf + β × ERP. Prices only systematic risk. See [[Valuation-Core/capm-model]].

**CRP** — Country Risk Premium. Added to ERP for non-US companies. See [[Quick-Reference/constants]].

## D

**DCF** — Discounted Cash Flow. The intrinsic value of a company = sum of all future FCF discounted to present. See [[Valuation-Core/dcf-implementation]].

**DDM** — Dividend Discount Model. Values a stock based on future dividends. Best for banks, utilities. See [[Research-Methods/institutional-overview]].

## E

**EBITDA** — Earnings Before Interest, Taxes, Depreciation, Amortization. Proxy for operating cash flow before capex. See [[Market-Benchmarks/sector-ev-ebitda]].

**Enterprise Value (EV)** — Market Cap + Debt − Cash. The total cost to acquire the company. See [[Quick-Reference/formulas]].

**ERP** — Equity Risk Premium. Expected excess return of stocks over risk-free rate. See [[Valuation-Core/capm-model]].

**EVA** — Economic Value Added. Net Operating Profit After Tax − (Capital × Cost of Capital). See [[Research-Methods/institutional-overview]].

## F

**FCF** — Free Cash Flow. Cash generated after maintaining capital assets. FCF = OperatingCF − CapEx. See [[Valuation-Core/dcf-implementation]].

**FCFF** — Free Cash Flow to Firm. Cash available to all capital providers (debt + equity). See [[Valuation-Core/dcf-implementation]].

**F-Score (Piotroski)** — 9-point quality score based on profitability, leverage, and efficiency. F ≥ 7 = strong. See [[Financial-Health/piotroski-f-score]].

**Football Field** — Chart showing valuation ranges from multiple methods. See [[Research-Methods/football-field]].

## G

**Gordon Growth Model** — Terminal value formula: TV = FCFₙ(1+g) / (WACC−g). See [[Valuation-Core/terminal-value]].

**GAAP XBRL** — Standardized format for digital financial reporting. SEC mandate since 2009. See [[Data-Layer/gaap-concepts]].

## I

**Intrinsic Value** — The "true" value based on fundamentals, not market price. Core of value investing. See [[Research-Methods/dcf-vs-comps]].

## L

**LBO** — Leveraged Buyout. Acquisition using significant debt. Used by Private Equity. See [[Research-Methods/lbo-and-sotp]].

**LTM** — Last Twelve Months. Standard period for comps analysis. See [[Market-Benchmarks/trading-comps]].

## M

**MDA** — Multiple Discriminant Analysis. Statistical technique used by Altman (1968) to derive Z-Score. See [[Financial-Health/altman-z-score]].

## N

**NAV** — Net Asset Value. Used for REITs and investment companies. See [[Research-Methods/lbo-and-sotp]].

## P

**P/E** — Price-to-Earnings ratio. Most widely used valuation multiple. See [[Market-Benchmarks/pe-ratios]].

**P/BV** — Price-to-Book Value. Used for banks, insurance, asset-heavy firms. See [[Market-Benchmarks/pe-ratios]].

## R

**Rf** — Risk-Free Rate. Typically 10-year government bond yield. See [[Quick-Reference/constants]].

**ROA** — Return on Assets = Net Income / Total Assets. See [[Financial-Health/piotroski-f-score]].

## S

**SOTP** — Sum-of-the-Parts. Breaks a conglomerate into segment values. See [[Research-Methods/lbo-and-sotp]].

## T

**Terminal Value** — Value of a company beyond the projection period. Usually 60-80% of DCF. See [[Valuation-Core/terminal-value]].

## W

**WACC** — Weighted Average Cost of Capital. Discount rate for DCF. Blends cost of equity and after-tax cost of debt. See [[Valuation-Core/wacc-theory]].

**Working Capital** — Current Assets − Current Liabilities. Used in Altman Z X₁ and Piotroski F #6. See [[Quick-Reference/formulas]].

## Z

**Z-Score** — see Altman Z-Score above.
