## How to Use

```
Implied EV = Target's EBITDA × Industry EV/EBITDA
Implied Equity Value = Implied EV − Net Debt
Implied Price per Share = Implied Equity Value / Shares Outstanding
```

## Interpretation Guidelines

| EV/EBITDA | Interpretation |
|-----------|----------------|
| > 30× | High-growth/premium; check for overvaluation or strong moat |
| 15-30× | Growth companies; typical for tech/healthcare |
| 8-15× | Mature industries; typical for consumer/industrial |
| < 8× | Cyclical/distressed; check for value trap or cyclical trough |

## Sector-Specific Gotchas

- **Banks/Insurance** — EBITDA is irrelevant; use P/E or P/BV (see [[Market-Benchmarks/pe-ratios]])
- **REITs** — Use P/FFO or P/AFFO instead of EV/EBITDA
- **Commodity firms** — Use normalized (through-the-cycle) EBITDA, not spot
